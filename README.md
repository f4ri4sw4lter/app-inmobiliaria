# app-inmobiliaria
Aplicacion web para Inmobiliaria Ferreyra


# Instalacion
1.Requisitos:
* node <= v18
* npm <= v9
2. Clonar el repositorio
3. Dentro del directorio ejecutar 
  ```
  npm install
  npm run build
  npm run start
  ```
4.En el navegador dirigirse a la direccion:

>http://localhost:3000/
